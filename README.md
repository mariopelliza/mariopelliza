## 🧰 Herramientas — reconocimiento y análisis de vulnerabilidades

| Herramienta | Función | Uso típico |
|---|---|---|
| **Nmap** | Escaneo de red | Descubrimiento de hosts, puertos, servicios y versiones (`-sV`, `-sC`, `-A`) |
| **Masscan / RustScan** | Escaneo rápido | Escaneo masivo de puertos en grandes rangos de IP |
| **Nikto** | Scanner web | Detecta configuraciones inseguras, CVEs y archivos expuestos en servidores HTTP |
| **Gobuster / FFUF** | Fuzzing | Enumeración de directorios, archivos y subdominios ocultos |
| **SQLMap** | Inyección SQL | Detección y explotación automática de SQLi |
| **WPScan** | WordPress | Enumeración de usuarios, plugins vulnerables y temas en sitios WP |
| **Shodan** | OSINT | Búsqueda de activos expuestos, puertos abiertos y banners en internet |
| **theHarvester** | OSINT | Recolección de emails, subdominios, IPs y metadatos de un objetivo |
| **Recon-ng** | OSINT | Framework modular de reconocimiento pasivo y activo |
| **SpiderFoot** | OSINT | Automatización de reconocimiento sobre dominios, IPs, emails y más |
| **Maltego** | OSINT | Visualización gráfica de relaciones entre entidades |
| **Burp Suite** | Web proxy | Intercepción, análisis y manipulación de tráfico HTTP/S |
| **Metasploit** | Explotación | Framework de explotación de vulnerabilidades conocidas |
| **BloodHound** | Active Directory | Mapeo de rutas de ataque y escalación de privilegios en AD |
| **CrackMapExec** | Post-explotación | Enumeración y movimiento lateral en redes Windows |
| **Impacket** | Protocolos Windows | Pass-the-Hash, PtT, DCOM, SMB, Kerberos |
